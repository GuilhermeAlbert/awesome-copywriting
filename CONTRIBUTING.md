# Contributing

Thank you for contributing. This list is curated and intentionally selective.

## Scope

This repository accepts contributions to:

1. Players (copywriters, strategists, or educators with documented impact)
2. Frameworks (named structures used to build persuasive copy)
3. Swipe files and archives (primary study references)
4. Books and learning resources

## Inclusion Criteria

### Players

A player should meet all conditions below:

1. Documented work or clear influence in the field
2. At least one primary public source (official site, book page, interview, or archived material)
3. A short factual summary of why the work matters

### Frameworks

A framework should include:

1. A recognized name used by more than one source
2. A concise explanation of structure and practical use
3. A source link when possible

### Resources

A resource should be:

1. Useful for study and analysis of real copy
2. Publicly accessible
3. Relevant to copywriting practice, not generic marketing news

## How To Open A PR

1. Fork the repository.
2. Edit `README.md` in `awesome-copywriting`.
3. Add the entry in the correct section.
4. Keep the description factual and short.
5. Add at least one primary link.
6. Open a pull request with a clear title and a short rationale.

## Writing Style

Use direct language:

1. Prefer concrete claims over hype.
2. Avoid promotional wording.
3. Keep one paragraph per player.
4. Use consistent formatting for links and headings.

## What Not To Add

Do not add:

1. Entries with no public source
2. Personal portfolios with no documented results or recognized work
3. Duplicated entries
4. Pure tool lists that are not directly related to copywriting study
